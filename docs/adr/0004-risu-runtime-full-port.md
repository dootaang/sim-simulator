# ADR 0004: Risu 카드 런타임 전체 이식 — "번역"에서 "격리 실행 + 엔진 소유"로

- 상태: 승인
- 결정일: 2026-07-14
- 적용 범위: 카드 표현·변수·스크립트 계층, 보안 경계, 카나리아 전략
- 상위 결정: ADR 0002를 **개정** (제품 정체성·엔진 소유 원칙은 유지, 호환 계층 전략만 교체)

## 맥락

ADR 0002의 B안은 "리스의 Lua/CBS/정규식을 실행하지 않고 의미를 우리 엔진으로 번역한다"였다.
전제는 *시뮬 로직만 번역하면 표현은 안전한 부분집합으로 충분하다*는 것이었다.

실측이 그 전제를 무너뜨렸다. 부분집합 이식의 실패가 반복됐고(맨 이름 이미지 소각, 감정 변형 미해석,
`#when`/`module_enabled` 양쪽 분기 유출), 결정적으로 용사여관의 감정 스프라이트 소멸 사례에서
**표현·변수·스크립트가 분리 불가능한 한 덩어리**임이 증명됐다: 카드의 정규식이 만드는 에셋 이름은
카드의 Lua 트리거가 채우는 CBS 변수(`npc_*_outfit`)로만 완성된다. 정규식·CBS·에셋 해석을 전부
고쳐도 Lua가 돌지 않으면 그 셋을 잇는 변수가 비어 파이프라인이 끊긴다.

부분집합 전략의 유지비는 "카드 생태계 전체가 테스트 케이스"인 두더지잡기이며, 이미 전체 이식
비용을 넘어섰다.

## 결정

1. **RisuAI의 카드 대면 런타임을 전체 이식한다** (기준 커밋 `eb7780b`, 로컬 클론 `C:\risu`,
   GPL-3.0 → 우리 GPL-3.0-or-later, 호환):
   - CBS 파서(`cbs.ts`), 표시 파서(`parser.svelte.ts`), 정규식(`scripts.ts`),
     트리거 v1/v2(`triggers.ts`), **Lua 런타임(`scriptings.ts`, wasmoon)**
   - 리스의 DB·Svelte 스토어 결합부는 **어댑터 계약**으로 갈아끼운다.
     이식 파일은 업스트림 경로를 미러링해(`packages/risu/src/port/*`) diff 추적을 가능하게 한다.
2. **엔진은 계속 우리 것이다.** 세이브·롤백·세션 저널·근거 기억·무결성은 럭키 엔진이 소유하고,
   컴파일된 카드에서는 결정론 사실(골드·시간·전투)도 엔진이 소유한다.
3. **"임의 실행 금지"의 정신은 계승한다.** Lua는 wasmoon(WASM 샌드박스) 안에서, 리스가
   `declareAPI`로 등록한 유한 API(실측 54개)의 **우리 화이트리스트 부분집합**만으로 실행한다:
   - 기본 차단: `request`(네트워크), `python`, `generateImage`
   - 옵트인 게이트: `LLMMain`/`axLLMMain`/`simpleLLM`(BYOK 비용 발생 — 사용자 승인 필요)
   - 이는 DESIGN.md의 기존 조항("코드 실행을 도입한다면 capability로 분리하고 기본 거부")의 이행이다.

## 이중 상태 정책 (이 결정의 유일한 신규 설계 난제)

카드 Lua는 chatVar(카드 변수)로 자기 상태를 가질 수 있다. 엔진 상태와의 충돌은 3모드로 가른다:

| 모드 | 상태 소유 | 엔진 역할 |
|---|---|---|
| 일반/호환 (기본) | 카드 Lua(chatVar) | 세이브·롤백·저널·기억 — 심판 없이 기록관 |
| 완전 시뮬 (컴파일 후) | 엔진 | 카드 Lua의 상태 변경 API를 읽기 전용으로 강등, 표시만 허용 |

핵심 이점: 세션 스냅샷이 이미 `cbsVariables`를 무결성 해시까지 포함해 저장하므로,
**카드 Lua의 상태까지 우리 체크포인트가 되감는다.** 카드 무수정으로 리스가 구조적으로 못 주는
롤백·저널·근거 기억을 제공한다. 제품 문구가 단순해진다: *"완벽 호환이 기본, 엔진 컴파일은 업그레이드."*

## 마일스톤과 새 북극성 시험

M-A `scripts.ts`(정규식) → M-B `cbs.ts` → M-C 트리거 v1/v2 → M-D **Lua** → M-E 모드 정책.
M-D의 합격선이 새 카나리아다: **용사여관을 컴파일 없이 열어 스프라이트·상태창·태그가 그냥 작동한다.**
통과 시 DOMINIUM·Tesselia 카나리아는 이 런타임의 부분집합이므로 사실상 함께 통과한다.

## 유지되는 것 / 감수하는 것

- 유지: 결정론 커널·모듈·컴파일러·조종석(HUD·결정 카드·로스터)·보안 척추 전부.
  `ysp-translate` 등 태그 번역기는 완전 시뮬 모드 전용으로 재배치(폐기 아님).
- 감수: 업스트림 추적 부담(커밋 고정 + 주기 diff), 어댑터 접합 공수(DB 결합 실측 108곳),
  번들 +271KB(wasmoon WASM — 기존 SQLite WASM 865KB보다 작음), Lua 실행 성능 측정 필요.
- 우리 자체 강화(정규식 시간·길이 예산, catastrophic 패턴 스킵, 최종단 sanitize)는
  이식 코드 **바깥의 래퍼**로 유지한다 — 업스트림 의미론을 오염시키지 않으면서 방어를 지킨다.
