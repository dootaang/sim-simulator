## 비교 불변식

- 같은 seed와 사건열은 같은 상태·로그를 만든다.
- 실패한 사건은 상태와 RNG를 소비하지 않는다.
- 기존 SimPack과 세션 파일은 migration을 거쳐 열린다.
- Risu 무편집 내보내기는 원본 바이트를 보존한다.
- 기억은 근거·유효 시점·공개 범위를 잃지 않는다.
- 화면 행동은 등록된 엔진 이벤트만 발생시킨다.
